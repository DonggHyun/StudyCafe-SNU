# Kiosk SNU
#### Study Cafe and U : 중앙정보처리학원 401호 팀 프로젝트

* 프로젝트 개요
* 사용 툴
  * IntelliJ
  * Java JDK 11.0.18
  * Apache Tomcat 8.5.88
  * MySQL 8.0.33
  * Spring 5.3.28
    
* Git 미포함 파일
  * src\main\resources\config\database.properties
    * 데이터베이스 연결을 위한 프로퍼티 파일
    * 데이터베이스 암호가 포함되어있으므로 git 저장소에 넣지 않음
    * jdbc.driverClassName
    * jdbc.url
    * jdbc.username
    * jdbc.password
    * jdbc.initialSize
    * jdbc.maxTotal
